# Smart Notes Summarizer

Smart Notes Summarizer is a web application that helps users quickly convert long notes or paragraphs into short, easy-to-read summaries. It is useful for students, professionals, and anyone who wants to save time while studying or reading.

---

## Features

- Summarizes long notes into short and clear points
- Fast and simple user interface
- Easy to use for study notes and revision
- Responsive design for desktop and mobile
- Built using Next.js and React

---

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Node.js

---

## Installation

Follow these steps to run the project locally:

1. Clone the repository

```bash
git clone https://github.com/Manish19hub/Smart-Notes-Summerizer.git
```

2. Move into the project folder

```bash
cd Smart-Notes-Summerizer
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```

5. Open your browser and visit

```text
http://localhost:3000
```

---

## Usage

1. Open the application in your browser.
2. Paste or type your notes into the input field.
3. Click the **Summarize** button.
4. Wait a few seconds for the app to generate a summary.
5. Read or copy the summarized text.

### Example

#### Input

```text
Artificial Intelligence is a branch of computer science that focuses on creating machines capable of performing tasks that normally require human intelligence. It includes learning, reasoning, and problem-solving.
```

#### Output

```text
Artificial Intelligence enables machines to perform tasks requiring human intelligence such as learning and problem-solving.
```
